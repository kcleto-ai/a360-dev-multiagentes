# Projeto-base (instalação de 1 clique)

Copie esta pasta como ponto de partida de um projeto novo (ou só o `.claude/settings.json`
pra dentro do seu repositório).

Quando a pessoa abrir esta pasta no **app Claude → aba Code (modo Local)**:
1. O Claude pede pra **confiar na pasta**.
2. Aparece um aviso pra **instalar o marketplace `a360` + o plugin `a360-dev-multiagentes`**.
3. Ela clica em **Instalar** — pronto, sem terminal, sem comando.

Depois é só digitar `/a360-dev-multiagentes` pra começar.
